# diskarte-site

**Public one-pager / marketing site** for Diskarte — an AI-native holdings operating system.

This repository is the public face of Diskarte: a focused, single-page presence describing
what Diskarte is and pointing the right audiences to the right next step. It is intentionally
lightweight and separate from the product (`diskarte-os`) and the institutional vault
(`diskarte-vault`).

## Layout

```
diskarte-site/
  src/      Site source (components, pages, styles)
  public/   Static assets served as-is (images, fonts, favicon)
```
